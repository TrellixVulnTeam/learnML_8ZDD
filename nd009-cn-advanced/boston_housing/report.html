编程练习3：定义衡量标准Â¶

如果不能对模型的训练和测试的表现进行量化地评估，我们就很难衡量模型的好坏。通常我们会定义一些衡量标准，这些标准可以通过对某些误差或者拟合程度的计算来得到。在这个项目中，你将通过运算决定系数 R2 来量化模型的表现。模型的决定系数是回归分析中十分常用的统计信息，经常被当作衡量模型预测能力好坏的标准。

R2的数值范围从0至1，表示目标变量的预测值和实际值之间的相关程度平方的百分比。一个模型的R2 值为0还不如直接用平均值来预测效果好；而一个R2 值为1的模型则可以对目标变量进行完美的预测。从0至1之间的数值，则表示该模型中目标变量中有百分之多少能够用特征来解释。模型也可能出现负值的R2，这种情况下模型所做预测有时会比直接计算目标变量的平均值差很多。

在下方代码的 performance_metric 函数中，你要实现：

  • 使用 sklearn.metrics 中的 r2_score 来计算 y_true 和 y_predict的R2值，作为对其表现的评判。
  • 将他们的表现评分储存到score变量中。

或

  • (可选) 不使用任何外部库，参考决定系数的定义进行计算，这也可以帮助你更好的理解决定系数在什么情况下等于0或等于1。

from sklearn.metrics import r2_score

def performance_metric(y_true, y_predict):
    """计算并返回预测值相比于预测值的分数"""
    
    score = r2_score(y_true, y_predict)

    return score

# 不允许导入任何计算决定系数的库

def performance_metric(y_true, y_predict):
    """计算并返回预测值相比于预测值的分数"""
    y_true1 = np.array(y_true)
    y_predict1 = np.array(y_predict)
    ytmp = np.mean(y_true1)
    SStot = np.sum((y_true1 - ytmp)**2)
    SSreg = np.sum((y_predict1 - ytmp)**2)
    SSres = np.sum((y_true1 - y_predict1)**2)
    
    score = 1.0 - SSres/SStot

    return score

编程练习 4：训练最优模型 （可选）Â¶

在这个练习中，你将需要将所学到的内容整合，使用决策树算法训练一个模型。为了得出的是一个最优模型，你需要使用网格搜索法训练模型，以找到最佳的 'max_depth' 参数。你可以把'max_depth' 参数理解为决策树算法在做出预测前，允许其对数据提出问题的数量。决策树是监督学习算法中的一种。

在下方 fit_model 函数中，你需要做的是：

  • 遍历参数‘max_depth’的可选值 1～10，构造对应模型
  • 计算当前模型的交叉验证分数
  • 返回最优交叉验证分数对应的模型

def cross_val_score(estimator, X, y, scoring = performance_metric, cv=3):
    """ 返回每组交叉验证的模型分数的数组 """
    split_tmp = len(X)/cv
    split0_X_train = X[:split_tmp* 2]    
    split0_X_test = X[split_tmp* 2:]
    split0_y_train = y[:split_tmp* 2]
    split0_y_test = y[split_tmp* 2:]
    
    split1_X_train = X[split_tmp:]    
    split1_X_test = X[:split_tmp]
    split1_y_train = y[split_tmp:]
    split1_y_test = y[:split_tmp]
    
    split2_X_train = X[:split_tmp].append(X[split_tmp* 2:])    
    split2_X_test = X[split_tmp:split_tmp* 2]
    split2_y_train = y[:split_tmp].append(y[split_tmp* 2:])
    split2_y_test = y[split_tmp:split_tmp* 2]
    
    estimator0 = estimator.fit(split0_X_train,split0_y_train)
    split0_y_predict = estimator0.predict(split0_X_test)
    split0_score = scoring(split0_y_test,split0_y_predict)
    
    estimator1 = estimator.fit(split1_X_train,split1_y_train)
    split1_y_predict = estimator1.predict(split1_X_test)
    split1_score = scoring(split1_y_test,split1_y_predict)
    
    estimator2 = estimator.fit(split2_X_train,split2_y_train)
    split2_y_predict = estimator2.predict(split2_X_test)
    split2_score = scoring(split2_y_test,split2_y_predict)
    
    
    
    scores = [split0_score,split1_score,split2_score]
    return scores


def fit_model(X, y):
    """ 基于输入数据 [X,y]，利于网格搜索找到最优的决策树模型"""
    if len(X) != len(y):
        return None
    
    tmpBest = 0.0
    tmpBestregressor = None
    for i in range(10):
        regressor = DecisionTreeRegressor(max_depth=i+1)
        tmp_mean_score = np.mean(cross_val_score(regressor, X, y))
        if  tmp_mean_score > tmpBest:
            tmpBest = tmp_mean_score
            tmpBestregressor = regressor
        
    #最优交叉验证分数对应的最优模型
    best_estimator = tmpBestregressor
    
    return best_estimator

模型健壮性Â¶

一个最优的模型不一定是一个健壮模型。有的时候模型会过于复杂或者过于简单，以致于难以泛化新增添的数据；有的时候模型采用的学习算法并不适用于特定的数据结构；有的时候样本本身可能有太多噪点或样本过少，使得模型无法准确地预测目标变量。这些情况下我们会说模型是欠拟合的。
